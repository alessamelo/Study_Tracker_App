# Study_Tracker_App
This project was developed using Kotlin and Android Studio. It consist on a study session tracker  with pixel art UI. 
The goal of this task was to implement an application using Json files as communication paths to the kt files

The project follow the following steps
'''
FormActivity
   ↓
Create User object
   ↓
Serialize → JSON
   ↓
Save JSON (SharedPreferences)
   ↓
Open SecondActivity
   ↓
Read JSON
   ↓
Deserialize → User object
   ↓
Display data
'''
